\subsection{Detailed Description}
This code compute:

Critical Curves and Caustics for

Tangential Direction, it uses \char`\"{}cc\_\-tan\_\-siep\char`\"{}

Radial Direction, it uses it uses \char`\"{}cc\_\-tan\_\-siep\char`\"{}

Curves of constant distortion in both planes. It uses \char`\"{}cc\_\-lw\_\-pos\_\-siep\char`\"{} and \char`\"{}cc\_\-lw\_\-neg\_\-siep\char`\"{}

to compile do: g++ -\/Wall -\/o exec\_\-siep \hyperlink{siep__example_8cpp}{siep\_\-example.cpp}

to run the program, there is two options:

the first option: we need a text file \char`\"{}input\_\-lens.txt\char`\"{} containing

iflag (integer to choice the parameterization, can be 1 or 2)

ellipticity value

Einstein Radius value

In the second option, in terminal to put the values of iflag, ellipticity value, Einstein Radius Value
